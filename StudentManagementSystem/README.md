# 🎓 Student Management System

A simple Student Management System developed using Python. This is a menu-driven console application that allows users to manage student records using CRUD (Create, Read, Update, Delete) operations.

---

##  Features

-  Add Student
-  View All Students
- Search Student by ID
-  Update Student Details
-  Delete Student
-  Display Total Students
-  Exit Application
