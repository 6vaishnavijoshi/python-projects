# Inventory Management System

A simple Inventory Management System built using Python. This is a menu-driven console application that allows users to manage products, update stock, and calculate inventory value using CRUD (Create, Read, Update, Delete) operations.

---

## Features

-  Add Product
-  View All Products
-  Search Product by ID
-  Update Product Details
-  Delete Product
-  Purchase Stock (Increase Quantity)
-  Sell Product (Decrease Quantity)
-  Display Total Products
-  Calculate Total Inventory Value
-  Exit Application
