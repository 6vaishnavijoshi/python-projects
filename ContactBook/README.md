# 📒 Contact Book

A simple Contact Book application built using Python. This project allows users to manage contacts through a menu-driven interface in the terminal.

## 🚀 Features

- ➕ Add a new contact
- 📋 View all contacts
- 🔍 Search for a contact
- ✏️ Update contact details
- 🗑️ Delete a contact
- 🚪 Exit the application

## 🛠️ Technologies Used

- Python 3
